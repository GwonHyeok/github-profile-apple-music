## Prerequisites

* [`Node.js / v16.15.0`](https://nodejs.org/en/)
* [`Yarn / v3.2.1`](https://yarnpkg.com/)

## Install

### set yarn version to yarn berry

```bash
yarn set version berry
```

### yarn install

```bash
yarn
```

## Packages

### backend

#### run backend locally

```bash
yarn start:backend
```

### web

#### run web locally

```bash
yarn dev:web
```

#### build web

```bash
yarn build:web
```

#### deployment web

```bash
yarn deploy:web
```
